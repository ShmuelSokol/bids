## If a bulk post goes wrong

Per-CIN rollback:
```
npx tsx scripts/_rollback-cin.ts --cinnum=0066229            # dry-run
npx tsx scripts/_rollback-cin.ts --cinnum=0066229 --execute  # commit
```

Reverses kbr/kae/kad deletes + k81/k80/ka9 status reverts + Supabase queue reset to `pending`. **Does NOT roll back k07 counters** (gaps are harmless) and does NOT undo the kaj→Shipped flip (warehouse physically shipped — flipping back is misleading).
